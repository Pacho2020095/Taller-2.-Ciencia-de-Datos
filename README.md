En el repositorio encontrarán los diferentes archivos que corresponden a los entregables del Taller 2.

La forma de entenderlos o leerlos según los requisitos del enunciado es la siguiente:

1. Diccionario de datos - apartamentos.html -> Preambulo. Datos entregados para hacer el taller.
2. Presentación_HabitAlpes_Taller2.pdf - > Presentación ejecutiva. Punto 6 del taller.
3. Taller2.pdg -> Preambulo. Enunciado del taller.
4. TallerFinal_v10.ipynb -> Notebook de Jupyter con el Entendimiento y preparación de los datos (Punto 1.), el Entrenamiento del modelo de Machine Learning (Punto 2), el Análisis cuantitativo de resultados del modelo (Punto 3) y el Análisis cualitativo de resultados del modelo (Punto 4)
5. Taller_2_Punto_5_CdDA.pdf -> Documento breve con el analisis de la generación de valor (Punto 5)
6. apartamentos.zip -> Preambulo. Base de datos suministrada para realizar el taller.
